開始使用
確認有安裝 node.js 與 npm

將專案 clone 到本地

在本地開啟之後，透過終端機進入資料夾，輸入：

npm install
安裝完畢後，繼續輸入：

npm run start
若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址

Listening on http://localhost:3000
若欲暫停使用

ctrl + c
